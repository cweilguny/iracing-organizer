# iRacing Organizer

iRacing Organizer is a single file iRacing season calendar that shows the whole season for multiple series at a glance.

View the current version at https://cweilguny.github.io/iracing-organizer/.


## Data

You have to enter the data manually. There is no magic and no iRacing API access built in. It's plain HTML and
JavaScript. It's an easy task and done in a couple of minutes, and only needed once every new season. I mainly race
GT and Open Wheel cars, so the version here will contain those series.


## Make it your own

Feel free to fork this repository or even just copy the index.html to fill it with your own series' data.


## Usage

Just drop the index.html on a webserver, create a GitHub page, or place it locally and open the file in your browser.
